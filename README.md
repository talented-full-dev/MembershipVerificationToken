## Install

```
git clone https://github.com/chaitanyapotti/ElectusProtocol.git
cd ElectusProtocol
npm install
```

## Contracts

The protocol level contracts use OpenZeppelin extensively for referencing standard EIPs

## Testing

To run all tests:

```
npm run test
truffle test
```
